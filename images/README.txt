Drop your photos in this folder:

  hero.jpg   →  1200 x 630  (social-share preview / Open Graph)
  story.jpg  →  ~800 x 1000 (4:5 portrait for the "Our Story" frame)

If a file is missing, the page shows an elegant labeled placeholder instead,
so nothing looks broken while you gather images.

Filenames are referenced in index.html (./images/...) — keep them the same,
or update the <img src="..."> paths if you rename.
